Halloween Gartic — Improved

Что нового:
- Нельзя начать игру пока не создана комната и не будет минимум 2 игроков.
- Теперь есть настройки комнаты: количество раундов, длительность хода, режим.
- Сервер управляет таймерами и авто-переходами между фазами.
- Клиентская рисовалка скорректирована: координаты правильно масштабируются под размер canvas (devicePixelRatio), рисование куда указывает курсор.
- Каждый шаг (рисование/угадывание) может выполнять только назначенный игрок — нельзя 'играть самому за всех'.

Как заменить старые файлы:
1) Распакуй ZIP или скопируй содержимое этой папки в место проекта.
2) Замени старые файлы (server.js, package.json, public/*) новыми.
3) Выполни в папке проекта:
   npm install
   npm start

Деплой на Render (быстро):
1) Залей эту папку в GitHub репозиторий (через веб-интерфейс или git).
2) На Render нажми New -> Web Service -> Connect GitHub -> выбери репозиторий.
3) Start command: npm start
